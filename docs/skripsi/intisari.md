# INTISARI

Ujian Tulis Berbasis Komputer – Seleksi Nasional Berdasarkan Tes (UTBK-SNBT) merupakan ujian *high-stakes* yang menentukan peluang siswa SMA untuk diterima di Perguruan Tinggi Negeri (PTN). Sebagian besar *platform* persiapan UTBK yang tersedia saat ini masih bersifat satu arah, yaitu hanya menampilkan soal dan skor akhir tanpa memberikan umpan balik yang kontekstual dan bertahap sesuai dengan proses berpikir siswa. Kondisi ini menciptakan *feedback gap* yang menghambat pemahaman konseptual dan mendorong budaya menghafal pola jawaban (*rote learning*). Di sisi lain, beberapa *platform* telah mengintegrasikan *chatbot* berbasis kecerdasan buatan, namun implementasinya masih terpisah dari konteks pengerjaan soal sehingga tidak mampu memberikan bimbingan secara *real-time*.

Penelitian ini bertujuan mengembangkan *platform* persiapan UTBK-SNBT berbasis web bernama Lexica yang mengintegrasikan *Intelligent Tutoring System* (ITS) berbasis *Large Language Model* (LLM) Llama-3.3 melalui Groq API. Sistem menerapkan metode *Socratic Scaffolding* bertingkat (SOCRATIC → HINT → SOLUTION) yang membimbing siswa secara bertahap tanpa langsung membocorkan jawaban. Penilaian kemampuan siswa menggunakan algoritma *Item Response Theory* (IRT) 1-Parameter Logistic (Rasch Model) yang memberikan estimasi kemampuan (θ) lebih adil dibandingkan skor persentase mentah. Selain itu, sistem dilengkapi dengan *Chancing Engine* untuk memprediksi peluang kelulusan pada program studi target, *Learning Path* personal berbasis analisis kelemahan per bab, dan dasbor analitik terpadu.

Pengembangan sistem menggunakan metode *Software Development Life Cycle* (SDLC) *Waterfall* dengan *framework* Next.js 16 (App Router), Prisma ORM, dan PostgreSQL. Desain antarmuka mengacu pada prinsip *Cognitive Load Theory* (Sweller, 1988) untuk meminimalkan beban kognitif yang tidak relevan (*extraneous cognitive load*). Pengujian dilakukan menggunakan *Black-Box Testing* dan kuesioner *System Usability Scale* (SUS). Hasil pengujian menunjukkan bahwa seluruh fitur utama berjalan sesuai rancangan dan skor SUS yang diperoleh berada pada kategori *acceptable*, sehingga *platform* Lexica layak digunakan sebagai sarana persiapan UTBK-SNBT yang interaktif, personal, dan terukur.

**Kata kunci:** *Intelligent Tutoring System*, *Socratic Scaffolding*, *Item Response Theory*, UTBK-SNBT, *Large Language Model*

# ABSTRACT

*The Computer-Based Written Examination – National Selection Based on Test (UTBK-SNBT) is a high-stakes examination that determines the opportunity for high school students to be admitted to state universities in Indonesia. Most currently available UTBK preparation platforms remain unidirectional, displaying only questions and final scores without providing contextual and graduated feedback aligned with students' thinking processes. This condition creates a feedback gap that hinders conceptual understanding and encourages rote learning patterns. While some platforms have integrated artificial intelligence chatbots, their implementation remains separate from the problem-solving context, rendering them unable to provide real-time guidance.*

*This research aims to develop a web-based UTBK-SNBT preparation platform called Lexica that integrates an Intelligent Tutoring System (ITS) powered by the Llama-3.3 Large Language Model (LLM) through Groq API. The system implements a tiered Socratic Scaffolding method (SOCRATIC → HINT → SOLUTION) that guides students progressively without directly revealing answers. Student ability assessment employs the Item Response Theory (IRT) 1-Parameter Logistic (Rasch Model) algorithm, which provides fairer ability estimates (θ) compared to raw percentage scores. Additionally, the system features a Chancing Engine for predicting admission probability to target study programs, a personalized Learning Path based on chapter-level weakness analysis, and an integrated analytics dashboard.*

*System development follows the Waterfall Software Development Life Cycle (SDLC) methodology using the Next.js 16 framework (App Router), Prisma ORM, and PostgreSQL. The user interface design adheres to Cognitive Load Theory principles (Sweller, 1988) to minimize extraneous cognitive load. Testing was conducted using Black-Box Testing and the System Usability Scale (SUS) questionnaire. Test results demonstrate that all core features function as designed and the obtained SUS score falls within the acceptable category, confirming that the Lexica platform is suitable for use as an interactive, personalized, and measurable UTBK-SNBT preparation tool.*

*Keywords: Intelligent Tutoring System, Socratic Scaffolding, Item Response Theory, UTBK-SNBT, Large Language Model*
